# Javaram_project
* 수행 목적 : 개발 능력 향상을 위한 Java 기반 Web project 수행  
 
* 주제 : 기본적인 CRUD 기능을 수행할 수 있는 Groupware 개발  
 *Groupware : 협업을 위해 필요한 다양한 비즈니스 기능을 하나의 플랫폼에서 지원하는 업무용 Software*  
 
* 시스템 구성 :
  * Local : 
    * Front-end : Vue 3.0  
        → Web-Socket
    * Back-end : Spring boot-Maven (Eclipse)
  * Cloud
    * Datebase : MariaDB(MySQL)
    * Mail-server

* 기능 구성 :  
  * Log-in
  * Mail
  * Notice
  * To-do
  * Web Chat
